Education

I wanted to do a cooperative education in electrical engineering that was primarily related to programming. MEA's Electrical Systems Research and Innovation Department was doing work on IoT systems, which was something I was interested in, so I decided to pursue a cooperative education at MEA.

The goal that has been set is to create projects related to the IoT system, which will be displayed using website applications and mobile applications. By evaluating the skills I already had. It is estimated that a maximum of 2 projects can be completed within 6 months.

Metropolitan Electricity Authority

Energy for city life, Energize smart living

Learning Phase

Mini Project 1

01 April 2024 - 30 April 2024

Front-end development of a massive open online course for mobile application and website

It's for a specialized knowledge course. Made for courses in specialized knowledge. This concept is a preliminary guideline for future platform courses that will use databases to make both systems work together. Overall, for this Mini project 1, mobile & web applications have been successfully created, which is considered the front-end of the user side.

Learning Phase

Mini Project 2

01 May 2024 - 31 May 2024

Development of a smart home system for device control and monitoring using node-red

Node-red dashboard for the built-in smart home IoT system can control devices and monitor device values ​​using the MQTT protocol, which works with the raspberry pi pico w board.

Development Phase

Project 1

01 June 2024 - 31 August 2024

Development of a cross-platform device control and monitor system for mobile and web applications

Utilized Flutter and Dart for the mobile application, and HTML, CSS, and JavaScript for the web platform. Both platforms were integrated via Firebase to enable seamless smart home device management.

Development Phase

Assigned Project 2

01 September 2024 - 31 September 2024

Development of a usage monitor system for electric vehicle charging stations aimed at sustainable energy usage

Utilized Flutter And Dart for building a responsive user interface and Firebase for real-time data management, enabling station managers to efficiently monitor and analyze station performance.

Development Phase

Extra Project 3

18 October 2024 - 6 December 2024

Development of a Web Application for Monitor Garbage Truck Charging Stations and Data Analysis

Utilized HTML, CSS, and JavaScript for the frontend, and Node.js with MySQL for backend development, enabling comprehensive and in-depth data analysis for efficient usage monitoring.
